= WeiXin

微信公众平台 开放消息接口 base on [sinatra](http://www.sinatrarb.com/)

== Installation

  gem install weixin
  
== Usage Examples

Config your api_key, api_secret and redrect_uri somewhere like development.rb.

  WeiXin::Config.token = "1234567890"
  WeiXin::Config.url = "somethinglooksstrageandhardtoremember"
